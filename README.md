# Aynekulu Full-Stack Portfolio

A **modern, interactive, and responsive developer portfolio** built using **HTML, Tailwind CSS, and JavaScript**.  
It showcases projects, skills, and a contact form with **dynamic features**.

---

## Features

- **Responsive design** with Tailwind CSS
- **Sticky navbar** with smooth scroll and active section highlight
- **Hero section** with typewriter effect
- **Skills section** with scroll animations
- **Projects section** with professional card layout
- **Interactive contact form** with validation
- **Back-to-top button** for easy navigation

---

## Technologies Used

- **Frontend**: HTML, Tailwind CSS, JavaScript
- **Backend (optional)**: Django (for full-stack integration)
- **Other Tools**: Git, GitHub
